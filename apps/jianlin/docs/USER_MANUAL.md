## 首頁管理

### 功能概述
首頁管理允許您：
- ✅ 新增/編輯/刪除首頁內容區塊
- ✅ 調整區塊順序
- ✅ 控制區塊顯示/隱藏
- ✅ 上傳和管理圖片

### 步驟 1: 進入首頁管理
在管理後台選單點擊 **「首頁管理」** → **「內容區塊」**

![首頁管理入口](screenshots/04-home-blocks-entry.png)
> **截圖說明:** 左側選單顯示「首頁管理」選項

### 步驟 2: 查看現有區塊
您會看到所有首頁內容區塊的列表

![區塊列表](screenshots/05-blocks-list.png)
> **截圖說明:**
> - **標題**: 區塊名稱
> - **類型**: 區塊類型（文字、圖片等）
> - **狀態**: 顯示 / 已隱藏
> - **操作**: 編輯、刪除、上移、下移按鈕

### 步驟 3: 新增區塊
點擊右上角 **「新增區塊」** 按鈕

![新增區塊按鈕](screenshots/06-add-block-button.png)
> **截圖說明:** 「新增區塊」按鈕位於列表右上方

### 步驟 4: 填寫區塊資訊
在編輯表單中填寫：

![編輯表單](screenshots/07-edit-form.png)
> **截圖說明:**
> 1. **標題**: 輸入區塊標題
> 2. **富文本內容**: 使用編輯器輸入內容
> 3. **圖片**: 上傳相關圖片（可選）
> 4. **顯示**: 勾選「顯示在前台」

### 步驟 5: 儲存區塊
填寫完成後，點擊底部 **「儲存」** 按鈕

![儲存按鈕](screenshots/08-save-button.png)
> **截圖說明:** 「儲存」按鈕在表單底部

### 步驟 6: 調整順序
返回列表頁，使用 **↑** **↓** 按鈕調整區塊順序

![調整順序](screenshots/09-reorder-blocks.png)
> **截圖說明:**
> - **↑ 按鈕**: 向上移動
> - **↓ 按鈕**: 向下移動

### 步驟 7: 儲存所有變更
調整完成後，點擊頁面頂部 **「儲存所有變更」**

![儲存所有變更](screenshots/10-save-all-changes.png)
> **截圖說明:** 「儲存所有變更」按鈕會保存排序和顯示狀態

## 關於頁面管理

### 功能概述
關於頁面提供 **5 種佈局模板**：
1. 📝 **純文字**: 僅標題和內容
2. 🖼️ **上圖下文**: 圖片在上，文字在下
3. ◀️ **左圖右文**: 圖片在左，文字在右
4. ▶️ **右圖左文**: 文字在左，圖片在右
5. 🖼️ **純圖片**: 僅顯示圖片

## 常見問題

### Q1: 上傳圖片失敗怎麼辦？
**A**: 檢查以下事項：
1. ✅ 圖片格式是否支援（JPEG、PNG、GIF、WebP）
2. ✅ 圖片大小是否超過 10MB
3. ✅ 網路連線是否正常
4. ✅ 是否已登入管理員帳號

### Q2: 為什麼我的變更沒有顯示在前台？
**A**: 請確認：
1. ✅ 已點擊「儲存」或「儲存所有變更」
2. ✅ 區塊的「顯示」選項已勾選
3. ✅ 清除瀏覽器快取（Ctrl+F5 或 Cmd+Shift+R）

### Q3: 如何刪除區塊？
**A**:
1. 在列表頁找到要刪除的區塊
2. 點擊「刪除」按鈕
3. 確認刪除
4. 點擊「儲存所有變更」

### Q4: 如何更改區塊順序？
**A**:
1. 在列表頁使用 ↑ ↓ 按鈕
2. 點擊向上 ↑ 將區塊往上移
3. 點擊向下 ↓ 將區塊往下移
4. 調整完成後點擊「儲存所有變更」

### Q5: 忘記密碼怎麼辦？
**A**: 請聯繫系統管理員重設密碼

### Q6: 圖片上傳後看起來很模糊？
**A**:
1. 請使用高解析度圖片（建議寬度 ≥ 1920px）
2. 避免上傳已經被壓縮過多次的圖片
3. 使用原始圖片而非截圖

### Q7: 如何在富文本中添加多個段落？
**A**:
在編輯器中按 **Enter** 鍵即可創建新段落

### Q8: 表格可以有多少行和列？
**A**:
- 理論上沒有限制
- 建議不超過 10 行 x 6 列以保持閱讀性
- 過大的表格建議使用 Excel 或 PDF 附件

### Q9: YouTube 視頻無法播放？
**A**:
1. ✅ 確認 YouTube URL 格式正確
2. ✅ 確認該視頻允許嵌入
3. ✅ 確認視頻沒有地區限制
4. ✅ 檢查網路連線

### Q10: 如何複製現有區塊？
**A**:
目前需要：
1. 點擊「編輯」打開現有區塊
2. 複製所有內容
3. 點擊「新增區塊」
4. 貼上內容
5. 修改標題
6. 儲存

*(未來版本將加入一鍵複製功能)*
